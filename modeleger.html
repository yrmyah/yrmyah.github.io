trayer :

Un systray simple et léger utilisable sur les bureaux qui n’en fournissent pas.
Toute la doc avec man trayer.
Je l’utilise avec la commande suivante, dans un script de lancement d’applications :

trayer --align center --edge top --transparent true --alpha 0 --tint black --heighttype pixel --height 22 --widthtype request --SetDockType true --SetPartialStrut false &

xbindkeys :

Comment appelle-t’on cela? Un gestionnaire (et daemon) de raccourcis clavier?
Très pratique, je l’utilise avec LWM, WMX et W9WM. Il est lancé dans un autostart et on définit les combinaisons comme cela:

"frankendmenu.sh"
Mod1 +  F1

"rofi -show run"
Mod1+shift + F1

"xterm -geometry 30x30 -e /home/yirmeyah/.config/nnn/plugins/launch"
Mod1 + F10

9menu :

Un menu/lanceur simple et configurable, qui n’est pas spécifique à 9wm ou w9wm. Les sous-menus sont possibles.
Pour l’utiliser, je crée un script que je nomme par exemple 9mainmenu.sh :

#!/bin/sh
9menu -label “MAIN MENU” -warp -popdown -teleport -bg “#1A1A1A” -fg “#7F7F7F” -font “-misc-fixed-medium-r-normal--16-------” -file “~/monmenu.txt”

Qui appelle un simple fichier texte, par exemple monmenu.txt :

 BAT         : acpi | dmenu -i -fn 'DejaVu Sans Mono-12' -nb '#000000' -nf '#7F7F7F' 
 -sb '#191919' -sf '#7F7F7F' 
 T42NOTES    : stterm -g 75x40+80+50 -e micro ~/t42notes.txt 
 MTM         : stterm -e mtm 
 CAL         : gsimplecal 
 GEANY       : geany 
 PCMAMFM     : pcmanfm 
 LUAKIT      : luakit 
 SCROT-DELAY : scrotdelay.sh 
 SCROT-SEL   : scrotsel.sh 
 ***
 LOGOUT      : loginctl terminate-session ${XDG_SESSION_ID-} 
 REBOOT      : systemctl reboot 
 SHUTDOWN    : systemctl poweroff

Et je le lance par un raccourci clavier défini dans .xbindkeysrc :

 "9mainmenu.sh" 
 Mod1 + a

mtm :

Un Multiplexeur de Terminal Micro, très léger, très simple (et quelque part limité) avec très peu de raccourcis clavier.
La combinaison de base est Ctrl+g + (touche). Mais cela peut-être modifié avant compilation. Ici le man et la Page github.
